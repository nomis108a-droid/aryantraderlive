# **App Name**: Aryan Gold FanHub

## Core Features:

- Authenticated Creator Onboarding: Secure Google sign-in integration and a detailed fan page registration form with image proof upload to Cloud Storage.
- Real-time Power Leaderboard: A dynamic, live-updating ranked list of all approved fan pages featuring real-time follower counts via Firestore onSnapshot.
- Milestone & Prize Tracking: Visual progress bars and achievement badges that track growth toward cash tiers and gift rewards.
- AI Content Optimization Tool: An AI-powered tool that analyzes recent live stream themes to suggest high-impact captions and trending hashtags for new Reels.
- Weekly Competition Arena: Submission portal for creators to link their best weekly Reels, featuring a countdown timer and a peer voting system.
- Central Admin Moderator Command: Password-protected dashboard for approving registrations, manually verifying followers, and managing prize payouts.
- Performance Statistics Dashboard: Individual creator profile pages showing engagement growth metrics and personal achievement history.

## Style Guidelines:

- Primary color: Radiant Gold (#FFD700) symbolizing success and the rewards for trading mastery.
- Background color: Deep Obsidian (#131314), providing a high-contrast foundation for the gold accents.
- Accent color: Neon Electric Blue (#00EEFC) used for interactive elements and neon-glow box-shadows.
- Headline font: 'Space Grotesk' for a high-tech, futuristic aesthetic; Body font: 'Inter' for crisp readability.
- Monospace stats: 'Source Code Pro' specifically for follower counts and data visualizations (Note: currently only Google Fonts are supported).
- Modern dark glassmorphism approach featuring backdrop-filter blur(12px) and subtle 1px translucent borders with rounded-xl corners.
- Smooth transitions for leaderboard rank shifts and gold gradient shimmer effects on milestone reward cards.
